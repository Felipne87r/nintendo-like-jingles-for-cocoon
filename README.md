Nintendo-Like Jingles for Cocoon Front-End

Custom jingles for games inspired by official 3DS and Wii banner jingles and Wii U splash/loading screens, for use in Cocoon launcher

How to use:

1. Get Cocoon: https://github.com/inssekt/CocoonFE

2. In Cocoon, go to settings > Library & Data > Jingles > Add new repository > Add the following:

* Repository Owner: felipne87r
* Repository Name: nintendo-like-jingles-for-cocoon

3. Go to any supported game, press Y or hold the icon > press Edit > Jingles > Press the magnifier icon > Download the desired jingle and save the edits
